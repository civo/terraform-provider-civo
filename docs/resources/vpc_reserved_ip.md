# civo_vpc_reserved_ip (Resource)

Provides a Civo VPC reserved IP to represent a publicly-accessible static IP address that can be mapped to one of your Instances or Load Balancer.

~> **Note:** This resource replaces the deprecated `civo_reserved_ip` resource and uses the VPC-scoped API endpoints.

## Example Usage

```terraform
resource "civo_vpc_reserved_ip" "www" {
    name = "nginx-www"
}

resource "civo_instance" "webserver" {
    # ...  removed for brevity
    reserved_ipv4 = civo_vpc_reserved_ip.www.ip
}
```
