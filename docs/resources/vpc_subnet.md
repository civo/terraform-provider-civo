# civo_vpc_subnet (Resource)

Provides a Civo VPC subnet resource. This can be used to create and delete subnets within a VPC network.

## Example Usage

```terraform
resource "civo_vpc_network" "example" {
    label = "example-network"
}

resource "civo_vpc_subnet" "example" {
    name       = "example-subnet"
    network_id = civo_vpc_network.example.id
}
```

- `name` (String) Name for the VPC subnet
