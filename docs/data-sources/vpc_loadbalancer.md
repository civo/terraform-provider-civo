# civo_vpc_loadbalancer (Data Source)

Get information on a VPC load balancer for use in other resources. This data source provides all of the load balancer's properties as configured on your Civo account.

An error will be raised if the provided load balancer name does not exist in your Civo account.

~> **Note:** This data source replaces the deprecated `civo_loadbalancer` data source and uses the VPC-scoped API endpoints.

## Example Usage

```terraform
data "civo_vpc_loadbalancer" "my-lb" {
    name   = "lb-name"
    region = "LON1"
}

output "civo_loadbalancer_output" {
    value = data.civo_vpc_loadbalancer.my-lb.public_ip
}
```
